# ATO API Documentation

Welcome to the ATO API documentation. This API allows you to manage orders, products, users, and pricing rules for the ATO system.

## Authentication

Most endpoints require authentication using an API key. Include your API key in the `api-key` header of your requests.

## Base URL

All API requests should be made to:

```
https://buswork.com.ar/api/
```

## Response Format

All responses are returned in JSON format.

## Error Handling

The API uses standard HTTP response codes to indicate the success or failure of requests. In case of an error, additional information may be included in the response body.
